刘德华（Andy Lau），1961年9月27日出生于中国香港，籍贯广东新会 [1] ，中国香港男演员、歌手、作词人、制片人。

1981年出演电影处女作《彩云曲》 [2] 。1983年主演的武侠剧《神雕侠侣》在香港获得62点的收视纪录 [3-4] 。1991年创办天幕电影公司 [5] 。1994年担任剧情片《天与地》的制片人 [6]

早年经历

2000年凭借警匪片《暗战》获第19届香港电影金像奖最佳男主角奖 [7] 。2004年凭借警匪片《无间道3：终极无间》获第41届台湾金马奖最佳男主角奖 [8] 。2005年获得香港UA院线颁发的全港最高累积票房香港男演员”奖 [9] 。 [13] 。 1985年发行首张个人专辑《只知道此刻爱你》 [14] 。1990年凭借专辑《可不可以》在歌坛获得关注 [15]

主要作品

1994年获得十大劲歌金曲最受欢迎男歌星奖。1995年在央视春晚上演唱其代表作《忘情水》 [16] 。2000年被《吉尼斯世界纪录大全》评为“获奖最多的香港男歌手” [17] 。2004年第六次获得十大劲歌金曲最受欢迎男歌星奖。2015年第5次登上央视春晚并演唱歌曲《回家的路》 [18] 。 演艺事业外，刘德华关心公益慈善。1994年创立刘德华慈善基金会 [19] 。2000年被评为世界十大杰出青年 [20] 。2005年发起亚洲新星导计划 [21] 。2008年被委任为香港非官守太平绅士 [22] 。2010年获得第12届世界杰出华人奖 [23] 。2016年连任中国残疾人福利基金会副理事长 [20] 。
